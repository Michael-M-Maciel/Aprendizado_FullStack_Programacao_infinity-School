Este é um parágrafo de exemplo para a atividade prática 01. Ele serve para demonstrar a manipulação do DOM usando JavaScript. Você pode usar este parágrafo para testar a alteração de texto, estilo, ou qualquer outra propriedade que desejar.

Sinta-se à vontade para experimentar diferentes métodos de seleção e manipulação de elementos do DOM. Lembre-se de que o DOM é uma representação em árvore dos elementos HTML, e você pode navegar por essa estrutura para acessar e modificar os elementos conforme necessário.
